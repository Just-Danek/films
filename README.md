# Films 🎬

Простое, функциональное веб-приложение для поиска фильмов с помощью фильтров, просмотра их детальной информации, добавления в избранное и наглядного сравнения их между собой. Данные подтягиваются через API Поиска кино.
Интерфейс приложения адаптирован под различные устройства и стилизован с использованием библиотеки VKUI.

## Ключевой функционал

- **Каталог фильмов** с реализацией **бесконечного скролла** (подгрузка по 50 элементов).
- **Продвинутая фильтрация**: по жанрам, диапазону рейтинга и годам выпуска.
- **Синхронизация с URL**: выбранные фильтры синхронизируются с URL-параметрами (search parameters), что позволяет делиться ссылками на конкретную выборку.
- **Детальная страница** с полным описанием, постером, жанрами и рейтингами.
- **Избранное**: добавление фильмов в личный список с подтверждением через модальное окно. Список сохраняется между сессиями.
- **Сравнение**: возможность выбрать до 2-х фильмов и сравнить их характеристики в удобной таблице (при добавлении третьего, первый автоматически удаляется).

## Стек технологий и инструменты

- **Frontend**: React, TypeScript
- **Сборщик**: Vite
- **Менеджер состояний**: Effector
- **Интерфейс и UI-кит**: VKUI, VKUI Icons, Lucide React
- **Роутинг**: React Router DOM
- **Полезные утилиты**: `react-intersection-observer` для бесконечного скролла

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте файл `.env` в корне проекта и добавьте ваш API ключ:
   ```env
   VITE_API_KEY="ВАШ_API_КЛЮЧ"
   ```
3. Запустите проект в режиме разработчика:
   ```bash
   npm run dev
   ```

## Что я изучил

1. **Работа с архитектурой Effector**:
    - Научился разделять бизнес-логику и UI-компоненты.
    - Освоил создание сторов (`createStore`), событий (`createEvent`) и эффектов для асинхронных запросов (`createEffect`).
    - Понял принцип чистоты редьюсеров (обработчиков `.on()`).
2. **VKUI и адаптивная верстка**:
    - Вычитал документацию VK UI.
    - Разобрался с построением макетов с помощью компонентов `SplitLayout`, `SplitCol`, `Epic` и `View`.
    - Изучил реализацию сложных UI-компонентов: модальные окна с контекстом (`ModalRoot`, `ModalCard`), адаптивные панели и списки.
3. **Продвинутый роутинг с React Router DOM**:
    - Научился связывать состояние приложения со строкой адреса (`useSearchParams`), чтобы фильтры не сбрасывались при обновлении страницы.
4. **Бесконечный скролл (Infinite Scroll)**:
    - Изучил работу с `Intersection Observer API` через библиотеку `react-intersection-observer` для оптимизации загрузки и рендера большого списка элементов.
